\section[xxx]{\textsc{ThePEG} version ThePEG d62cc40b1b9f () \cite{ThePEG} Run Information}
Run name: \textbf{LHC-S100100-mY_100_mX_100-2}:\\
This run was generated using Herwig~\cite{Bahr:2008pv} and the following models:
\begin{itemize}
\item BSM physics was produced using the algorithm of \cite{Gigg:2007cr,Gigg:2008yc}
\item In the running of $\alpha_{EM}$, the parametrization of H.~Buckhardt et al. was used. See \cite{KLEISSCERN9808v3pp129}.
\item The underlying event was simulated with an eikonal model for multiple partonic interactions.Details can be found in Ref.~\cite{Bahr:2008dy,Bahr:2009ek}.
\item The weak decay of baryons containing a heavy quark used form factors from \cite{Cheng:1995fe,Cheng:1996cs}.
\end{itemize}
